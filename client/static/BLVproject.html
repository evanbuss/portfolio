Project Beat Las Veags

An NFL analytics website. I’ve developed a machine-learning algorithm that predicts football games (70.31% in 2014 season). Along with predicting games I have used statistical methods learned in classes at Yale University to break down historical data from 1985 to present day. I’ve used this data to develop interactive features that are visually appealing to users. The project is built with the MEAN stack and is going to be deployed on an Amazon EC2 server before the start of the 2015 NFL season.
